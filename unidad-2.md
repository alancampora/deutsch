# Unidad 2

## Posesivos
Cada persona tiene su posesivo y se declinanaran dependiendo del genero 
del sustantivo. 

| Persona        |          | 
| ---------------|:--------:|
| ich            | mein/e   |
| du             | dein/e   |
| es/er          | sein/e   |
| sie            | ihr/ihre |
| wir            | unser/e  |
| ihr            | euer/e   |
| sie            | ihr/e    |
| Sie            | Ihr/e    |

> Was ist *deine* Telefonummer(f) ? 
> Das ist *mein* Auto(n)
> Wie ist *Ihre* Addresse(f) ?
