## Fragen 

### W-Frage
El verbo siempre en posicion 2.

| Position | 1 | 2  | 3 | 
|:--------:|:----:|:------:|:----:|
|          | Woher| kommen | Sie ?|
|          | Was  | bist   | du von Beruf?|
|          | Wie  | heissen | Sie ? |


### Preguntas comunes
#### Woher (de donde)
* Wo wonht er? 
* Wo wonhen Sie?

#### Wohin = hacia donde 
* Wohim geht er? 
* Wohim gehen Sie? 
* Wohim gehts du? 

#### Wann = cuando 
* Wann ist dein geburstag? 



### Ja/Nein Frage
